手動配置Vite + Vue + Electron 開發環境

手動配置的意義是，不讓vite生成一堆檔案結構，一步步慢慢手動建構，需要什麼就自己設定的那種。

技術棧

  • 開發環境工具：Vite
  • 桌面框架：Electron
  • UI框架：Vue

建置步驟

初始化起手

Vite - 開發伺服器 和 打包工具

  • Vite 也有快速建置的指令
  • Vite 是預設運行根目錄的 index.html

Vite 的運行

  1. 可以在package.json裡面寫script，這樣就可以用npm run dev 運行：
  2. 使用npx vite dev運行。

Vue - 前端框架

@vitejs/plugin-vue - Vite 處理 Vue 文件的插件

  • 官方文件 Vue - @vitejs/plugin-vue

然後寫進vite.config.js，這也是按照官方GitHub文檔

Electron - 桌面應用框架

  • 安裝
  • 在package.json配置入口
  • 按照文件在main.js建立簡單的主進程配置
  • 運行方式是vite跟electron都要運行
